## 文件说明

### autoplay.js
1.0 精简版脚本。

适用场景：

- 只需要在视频页自动下一集
- 不需要课程扫描、小节定位、悬浮球控制等附加功能

主要功能：

- 检测视频播放结束
- 自动进入下一课 / 下一集
- 逻辑简单，适合轻量使用

### StudyAids.js
3.1.6 完整版脚本。

适用场景：

- 需要自动定位未达标课程
- 需要自动定位未完成小节
- 需要视频自动下一课
- 需要自动恢复播放
- 需要静音
- 需要跨域同步暂停 / 启动状态
- 需要悬浮球统一控制

主要功能：

- 自动打开第一门未达标课程
- 自动打开第一个未完成小节
- 视频结束自动下一课
- 视频暂停自动恢复播放
- 视频静音
- 跨域同步配置
- 悬浮球可拖动
- 开始脚本 / 暂停脚本即时生效

---

## 使用建议

- 只需要“自动下一集” → 使用 `autoplay.js`
- 需要完整学习辅助流程 → 使用 `StudyAids.js`
