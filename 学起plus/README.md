## 功能简介

完整版脚本当前支持以下功能：

- 自动定位未达标课程
- 自动定位未完成小节
- 视频结束自动下一课
- 视频暂停自动恢复播放
- 低音量模式（替代脚本静音）
- 悬浮球控制面板
- 跨页面设置同步

## 功能说明

### 1. 自动定位课程

在课程列表页自动扫描未达标课程，并可自动打开第一门目标课程。

### 2. 自动定位小节

在课程目录页自动扫描未完成小节，并可自动进入第一个未完成小节。

### 3. 自动下一课

当前视频播放结束后，脚本会自动尝试进入下一课。

### 4. 自动恢复播放

当视频意外暂停时，脚本会尝试自动恢复播放。

### 5. 低音量模式

为避免脚本直接静音可能影响后台播放稳定性，当前版本使用低音量模式替代静音策略。

### 6. 悬浮球控制

页面右侧提供悬浮球，可随时打开控制面板，手动开关功能。

- 自动下一课触发稳定性

- 页面控制逻辑的可开关化
