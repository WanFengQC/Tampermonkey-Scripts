# 学起plus脚本说明

这个目录用于存放 **学起plus 相关脚本**，当前主要维护的是完整版学习助手脚本。

---

## 当前文件

### StudyAids.js

- **脚本名称**：学起plus学习助手
- **当前版本**：`3.2.5`
- **脚本类型**：完整版
- **适用场景**：学起平台课程列表页、课程目录页、视频播放页

**GreasyFork 安装地址：**  
[学起plus学习助手 3.2.5](https://greasyfork.org/zh-CN/scripts/571362)

## 功能概览

当前版本支持以下核心功能：

- 自动定位未达标课程
- 自动定位未完成小节
- 视频结束自动下一课
- 视频暂停自动恢复播放
- 低音量模式（替代脚本静音）
- 悬浮球控制面板
- 跨页面设置同步

---

## 功能说明

### 自动定位课程

在课程列表页自动扫描未达标课程，并可自动打开第一门目标课程。

### 自动定位小节

在课程目录页自动扫描未完成小节，并可自动进入第一个未完成小节。

### 自动下一课

当前视频播放结束后，脚本会自动尝试进入下一课。

### 自动恢复播放

当视频意外暂停时，脚本会尝试自动恢复播放，以减少后台切换带来的中断。

### 低音量模式

为避免脚本直接静音可能影响后台播放稳定性，当前版本使用 **低音量模式** 替代传统静音策略。

### 悬浮球控制

页面右侧提供悬浮球，可随时打开控制面板，灵活开关各项功能。

## 使用方式

1. 安装 Tampermonkey
2. 导入 `StudyAids.js`
3. 打开学起相关页面
4. 通过右侧悬浮球打开控制面板
5. 根据需要开启或关闭功能
